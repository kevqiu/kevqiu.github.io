MEDIASAURUS

The inspiration and motiviation behind Mediasaurus came from the need to optimize Netflix and TV time. People would spend countless minutes pondering which show or movie to watch next when that precious time could be used for media consumption instead. Mediasaurus helps solve this issue by recommending users movies depending on their genre and movie preferences. Each user is catered with a personalized list of movies which dynamically updates based off their recent 'likes'.

This web application was built using the MEAN stack (MongoDB, Express.js, AngularJS, Node.js). Our team of six decided on this tech stack because we wanted to use something modern and the flexibility of this stack allowed us to realize our product.

My role on Mediasaurus was in front end development. I worked on setting up website structure, designing UI/UX, implementing MVC, and integrating back end endpoints. The tools I used for these tasks were HTML, CSS with Bootstrap, and AngularJS.

HOME SECURITY SYSTEM

The Home Security System was built as my team's final Computer Engineering Design Studio Project. The project utilized a combination of hardware and software to achieve an IoT-based system.

Using an array of sensors connected to a master-slave Arduino setup using I2C, we were able to model a home security system with a door status checker, laser tripwires, motion detection, alarms, and timed lighting. We created an Android application with a simple UI to allow us to monitor, set, and trigger each individual component of the security system. Using a Raspberry Pi as our central server, we were able to relay data between the Arduinos and the Android app.

I lead in the construction of the model house and the development of the Android application. I was responsible for building the model house, selecting the sensor array, and wirnig everything together. For the Android application I lead in designing the layout the UI and building the core functionality for instruction handling and sending.
